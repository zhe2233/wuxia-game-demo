# wuxia-game-demo

由华为 CodeArts Agent AI 生成的武侠主题内倾思维（Ti）流派趣味测鉴网页，纯前端实现，开箱即用。

## 🎯 项目介绍
一个武侠风格的心理测试单页应用，通过20道选择题测评用户的思维模式，匹配对应的武侠人物流派（如独孤求败式极境归真、令狐冲式随性无羁等），完整支持答题流程、结果计算、移动端响应式适配。

## ✨ 核心特性
- 纯前端实现：原生 HTML/CSS/JavaScript，无需后端依赖
- 完整交互：20道题目、进度条、上/下一题导航、结果匹配
- 响应式设计：完美适配PC端和移动端
- 武侠主题UI：复古国风视觉，沉浸式体验

## 🚀 快速使用
1.  直接下载 `wuxia_test.html` 文件
2.  用浏览器打开即可开始测试，无需安装任何环境
